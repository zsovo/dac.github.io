PyProphet

PyProphet: Semi-supervised learning and scoring of OpenSWATH results. PyProphet is a Python re-implementation of the mProphet algorithm optimized for SWATH-MS data acquired by data-independent acquisition (DIA). The algorithm was originally published in and has since been extended to support new data types and analysis modes.
